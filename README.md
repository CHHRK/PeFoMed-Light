## Dataset Description

The dataset consists of chest X-ray report text used for training and evaluating the PeFoMed-Light model.

Structure:
- train/reports.csv
- val/reports.csv
- test/reports.csv

Each CSV contains:
- image_id: Unique identifier for the X-ray image
- report: Ground-truth radiology report text
- split: Dataset split (train/val/test)

The dataset is small in size and contains no personally identifiable information.
